# Jogo-Genius-FPGA
Universidade Federal de Santa Catarina

Semetre 2022/2

Erik Orsolin de Paula

O projeto final da disciplina de Circuitos e Técnicas Digitais (EEL5105) consiste na implementação de um circuito na placa de desenvolvimento DE2 fazendo uso das estruturas
e conhecimentos obtidos durante o curso. O circuito vai implementar a um jogo interativo similar ao jogo Genius.
